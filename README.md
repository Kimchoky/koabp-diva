# Koabp-DIVA

KOABP 제품군의 생산 직후, 반자동화된 검사를 위한 데스크탑 앱.

프로젝트 코드네임인 DIVA는 "Device Imprinting and Verifying Application"의 의미를 가짐 

## Stack
- Nextron (Electron + Next.js)
- noble library
- Claude :)


### 개발환경 설정

Hot-reload를 지원함.

```shell
# 의존 라이브러리 설치
$ npm install

# 개발 앱 실행 
$ npm run dev
```

### 배포용 앱 빌드
```shell
# production build
$ npm run make
```
